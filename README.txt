README HOW TO RUN THE APPLICATION

REQUIREMENTS:
Latest Node.js Version
Latest Python version
Have Metamask in browser

STEPS:
1) Open gitbash terminal in the 'lottery-dapp' folder
2) run the following lines in the console:
npm i
npm run compile
npm run dev

3) Go to http://localhost:3000/ in the browser



Please refer to video demonstration on how to use the application

NOTE: Since it is hardcoded on which wallet is the admin, the admin buttons PickWinner and PayWinner cannot be used, but the playgame and connect wallet button is usable.
